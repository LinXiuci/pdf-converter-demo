# 前端实现 PDF 转换器

## 实现方式

- ~~下载并安装 LibreOffice
- 使用 Koa 作为后台应用框架：

```
  koa
  koa-cors
  koa-multer
  koa-router
  koa-static
  nodemon
  office-to-pdf
```

- 使用 React Hooks 作为前台应用框架

```
  vite
  antd
  axios
  react
```

- 运行

```
后台：
  yarn
  yarn dev

前台：
  yarn
  yarn dev
```
